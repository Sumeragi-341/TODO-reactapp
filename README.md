# React TODOアプリ

## 概要
Reactを使用してシンプルなTODOアプリを開発しました。

## 機能
- タスクの追加 / 削除 / 編集
- 完了状態の切り替え
- 期限設定機能
- フィルター機能（全て / 未完了 / 完了）
- ローカルストレージによるデータ保存

## 使用技術
- React（useState / useEffect）
- JavaScript
- CSS

## 工夫した点
- 状態管理（useState）と副作用（useEffect）を意識した設計
- ユーザーが直感的に操作できるUIを意識
- データをローカルストレージに保存することで、リロード後も状態を保持

## 今後の課題

- デザインの改善
- 外部APIとの連携
